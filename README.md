# 作业统计系统

一个基于网页的作业进度跟踪工具，可以创建作业本、记录完成情况，并以座位式布局展示页面状态。

## 功能特点

- 📚 **创建作业本**：自定义作业本名称和总页数
- 🎯 **进度跟踪**：实时显示剩余页数和完成进度
- 🎪 **座位式布局**：类似电影票座位的页面网格展示
- 🟢 **状态标记**：已完成页面显示为绿色，未完成显示为灰色
- ✏️ **编辑功能**：点击编辑按钮可修改页面状态
- 📥 **数据导入导出**：支持JSON格式数据备份和恢复
- 📱 **响应式设计**：支持桌面和移动设备

## 使用方法

### 基本操作
1. **创建作业本**：点击"新建作业本"按钮，输入名称和总页数
2. **查看详情**：点击作业本卡片进入详情页面
3. **编辑页面状态**：点击右上角"编辑"按钮，然后点击页面格子修改状态
4. **跟踪进度**：顶部显示剩余页数和完成进度

### 数据管理
- **导出数据**：点击"导出数据"按钮下载JSON格式备份文件
- **导入数据**：点击"导入数据"按钮选择之前导出的JSON文件

## 部署到GitHub Pages

1. 将项目文件上传到GitHub仓库
2. 在仓库设置中启用GitHub Pages
3. 选择部署分支（通常是main或gh-pages）
4. 访问 `https://[用户名].github.io/[仓库名]` 即可使用
